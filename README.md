# News_Aggregator

## Overview

News Aggregator is a web application that collects and displays the latest news articles from various sources in one place. The application allows users to browse news by category, view article details, and stay updated with current events through a clean and user-friendly interface.

## Features

* View latest news headlines from multiple sources
* Browse news by categories
* Search for news articles
* Responsive and user-friendly interface
* Real-time news updates
* Organized news display with article images and descriptions

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* MongoDB

## Project Structure

* **public/** – Static files such as CSS, JavaScript, and images
* **src/** – Application source code
* **template/** – HTML templates and views
* **server.js** – Main server file

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/karishma-yadav11/News_Aggregator.git
   ```

2. Navigate to the project directory:

   ```bash
   cd News_Aggregator
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the application:

   ```bash
   npm start
   ```

5. Open your browser and visit:

   ```
   http://localhost:3000
   ```
